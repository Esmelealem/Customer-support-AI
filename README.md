# Customer Support AI

AI-powered customer support backend built with:
- Spring Boot
- OpenAI (chat + embeddings)
- PostgreSQL + pgvector
- RAG (Retrieval-Augmented Generation)

## Features
- Order status lookup
- Knowledge base search
- Vector similarity search
- Admin ingestion endpoint
- Offline & batch ingestion

## Tech Stack
- Java 21+
- Spring Boot
- PostgreSQL (pgvector)
- Docker
- OpenAI API
